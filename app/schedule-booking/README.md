# Schedule Booking Project

This project is a web application that displays available schedules and allows users to make reservations. It consists of a simple interface where users can view time slots and click a "Reserve" button to book their desired time.

## Project Structure

```
schedule-booking
├── src
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   ├── schedule.js
│   │   └── reservation.js
│   ├── components
│   │   ├── Schedule.js
│   │   └── ReservationButton.js
│   └── index.html
├── package.json
└── README.md
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd schedule-booking
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

1. Open `src/index.html` in your web browser to view the application.
2. The available schedules will be displayed on the page.
3. Click the "Reserve" button next to your desired time slot to make a reservation.
